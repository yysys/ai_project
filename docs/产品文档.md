# 猪了个猪 - 抖音小程序产品文档

## 1. 产品概述

### 1.1 产品简介
"猪了个猪"是一款轻松休闲的点击类抖音小程序，游戏场景中，中间有一个菜狗，周围有一群狼。玩家点击狼，狼会沿着对角线方向（左上、左下、右上、右下）跑动，目标是保护菜狗不被狼抓到。游戏采用可爱的卡通画风，为玩家带来愉悦的游戏体验。

### 1.2 产品定位
- **目标用户**：18-35岁的休闲游戏玩家
- **游戏类型**：休闲益智类点击游戏
- **平台**：抖音小程序
- **核心玩法**：点击 + 对角线跑动 + 保护菜狗

### 1.3 产品特色
- 可爱的卡通画风，吸引年轻用户
- 简单易上手，点击即可操作
- 策略性玩法，需要思考点击顺序
- 丰富的关卡设计，保持游戏新鲜感
- 社交分享功能，增加传播性
- 轻松治愈的游戏体验

## 2. 游戏玩法

### 2.1 核心玩法
- 屏幕中间有一个菜狗（主角）
- 周围有一群狼（敌人）
- 玩家点击狼，狼会沿着对角线方向跑动
- 四个运行方向：左上、左下、右上、右下
- 狼会自动追踪菜狗，试图抓到它
- 玩家需要策略性地点击狼，让狼跑出屏幕
- 保护菜狗存活即可过关

### 2.2 游戏规则
1. **基本规则**
   - 点击狼，狼开始沿对角线跑动
   - 狼的运行方向：左上、左下、右上、右下
   - 狼跑动速度固定，不可改变
   - 狼跑动到屏幕边界会停止
   - 狼碰到菜狗会伤害菜狗

2. **胜利条件**
   - 菜狗存活到关卡结束
   - 所有狼都被赶出屏幕
   - 完成关卡设定的任务目标

3. **失败条件**
   - 菜狗被狼抓到（生命值归零）
   - 超过时间限制
   - 菜狗生命值不足

### 2.3 关卡设计
- **初级关卡**：狼数量少，菜狗生命值高
- **中级关卡**：狼数量增加，菜狗生命值降低
- **高级关卡**：狼数量多，狼速度更快
- **特殊关卡**：限时模式、生存模式、挑战模式

### 2.4 角色类型
1. **菜狗（主角）**
   - 位于屏幕中心
   - 有生命值系统
   - 被狼碰到会受伤
   - 可以使用保护道具

2. **普通狼**
   - 基础狼，点击即可跑动
   - 速度中等
   - 无特殊能力

3. **快速狼**
   - 跑动速度更快
   - 更难控制
   - 需要快速反应

4. **协作狼**
   - 会与其他狼协作
   - 从不同方向包围菜狗
   - 更难对付

## 4. 功能模块

### 4.1 游戏核心功能
- **角色生成**
  - 在屏幕中心生成菜狗
  - 随机生成狼群位置
  - 根据关卡难度调整狼数量

- **点击检测**
  - 精确的点击区域检测
  - 防止重复点击
  - 点击反馈效果

- **跑动逻辑**
  - 对角线跑动算法
  - 四个方向：左上、左下、右上、右下
  - 碰撞检测
  - 边界检测

- **游戏状态管理**
  - 开始、暂停、继续、结束
  - 游戏进度保存
  - 关卡解锁状态

### 4.2 辅助功能
- **提示功能**
  - 显示最佳点击顺序
  - 高亮显示目标狼
  - 限制使用次数

- **重置功能**
  - 重置当前关卡
  - 限制使用次数

- **慢动作功能**
  - 减慢狼跑动速度
  - 方便观察和操作

### 4.3 保护功能
- **护盾道具**
  - 为菜狗提供临时保护

- **障碍道具**

  - 阻挡狼的路径
  - 持续时间：10秒

- **陷阱道具**
  - 在指定位置放置陷阱
  - 狼碰到会被困住
  - 持续时间：8秒

## 5. 关卡设计

### 5.1 关卡类型
1. **普通关卡**
   - 基础关卡，引导玩家熟悉玩法
   - 狼数量：3-8只
   - 菜狗生命值：3点
   - 目标：保护菜狗存活

2. **生存关卡**
   - 狼数量增加，生存挑战
   - 狼数量：8-15只
   - 菜狗生命值：2点
   - 目标：保护菜狗存活

3. **限时关卡**
   - 时间限制的关卡
   - 狼数量：10-20只
   - 时间：30秒-2分钟
   - 目标：在规定时间内保护菜狗

4. **挑战关卡**
   - 高难度关卡
   - 狼数量：20-40只
   - 菜狗生命值：1点
   - 目标：完成复杂任务

### 5.2 关卡元素
1. **菜狗**
   - 位于屏幕中心
   - 有生命值系统
   - 可以使用保护道具

2. **狼**
   - 普通狼
   - 快速狼
   - 协作狼

3. **道具**
   - 护盾道具
   - 障碍道具
   - 陷阱道具

### 5.3 关卡难度曲线
- **第1-10关**：新手引导，熟悉基本玩法
- **第11-30关**：引入快速狼，增加难度
- **第31-60关**：引入协作狼，丰富玩法
- **第61-100关**：限时关卡，考验反应速度
- **第101关+**：挑战关卡，考验策略思考
