# 拯救菜狗 - 抖音小程序产品文档

## 1. 产品概述

### 1.1 产品简介
"拯救菜狗"是一款轻松休闲的点击类抖音小程序，游戏场景中，中间有一个菜狗，周围有一群狼。玩家点击单位，单位会沿着对角线方向（左上、左下、右上、右下）跑动，目标是让菜狗跑出屏幕边界。游戏采用可爱的卡通画风，为玩家带来愉悦的游戏体验。

### 1.2 产品定位
- **目标用户**：18-35岁的休闲游戏玩家
- **游戏类型**：休闲益智类点击游戏
- **平台**：抖音小程序
- **核心玩法**：点击 + 对角线跑动 + 拯救菜狗

### 1.3 产品特色
- 可爱的卡通画风，吸引年轻用户
- 简单易上手，点击即可操作
- 策略性玩法，需要思考点击顺序
- 丰富的关卡设计，保持游戏新鲜感
- 轻松治愈的游戏体验

## 2. 游戏玩法

### 2.1 核心玩法
- 屏幕中间有一个菜狗（主角）
- 周围有一群狼（敌人）
- 玩家点击单位，单位会沿着对角线方向跑动
- 四个运行方向：左上、左下、右上、右下
- 玩家需要策略性地点击单位，让菜狗跑出屏幕
- 菜狗跑出屏幕即获胜

### 2.2 游戏规则
1. **基本规则**
   - 点击单位，单位开始沿对角线跑动
   - 单位的运行方向：左上、左下、右上、右下
   - 单位跑动速度固定，不可改变
   - 单位跑动到屏幕边界会停止

2. **胜利条件**
   - 菜狗跑出屏幕边界

3. **失败条件**
   - 超过时间限制
   - 菜狗没有在规定时间内跑出屏幕

### 2.3 关卡设计
- **初级关卡**：狼数量少
- **中级关卡**：狼数量增加
- **高级关卡**：狼数量多
- **特殊关卡**：限时模式、生存模式、挑战模式

### 2.4 角色类型
1. **菜狗（主角）**
   - 位于屏幕中心
   - 点击后沿对角线跑动
   - 跑出屏幕即获胜

2. **狼（敌人）**
   - 基础狼，点击即可跑动
   - 速度中等
   - 无特殊能力

## 4. 功能模块

### 4.1 游戏核心功能
- **角色生成**
  - 在屏幕中心生成菜狗
  - 随机生成狼群位置
  - 根据关卡难度调整狼数量

- **点击检测**
  - 准确的点击区域检测
  - 防止重复点击
  - 点击反馈效果

- **跑动逻辑**
  - 对角线跑动算法
  - 四个方向：左上、左下、右上、右下
  - 边界检测

- **游戏状态管理**
  - 开始、暂停、继续、结束
  - 游戏进度保存
  - 关卡解锁状态

### 4.2 辅助功能
- **提示功能**
  - 显示最佳点击顺序
  - 高亮显示目标单位
  - 限制使用次数

- **重置功能**
  - 重置当前关卡
  - 限制使用次数

- **慢动作功能**
  - 减慢单位跑动速度
  - 方便观察和操作
  - 限制使用次数

## 5. 技术规格

### 5.1 游戏棋盘
- **网格尺寸**：14列 × 14行
- **单元格尺寸**：18px × 18px
- **单元格类型**：
  - 横向单元格（tile-h）：占据2列
  - 纵向单元格（tile-v）：占据2行
- **棋盘旋转**：45度
- **背景颜色**：#dcedc8（浅绿色）

### 5.2 单位设计
- **菜狗**：
  - 尺寸：占据2列（tile-h）
  - 颜色：金色边框 #FFD700
  - 光晕效果：rgba(255, 215, 0, 0.6)
  
- **狼**：
  - 尺寸：tile-h 或 tile-v
  - 颜色：灰色 #64748b
  - 无特殊效果

### 5.3 按钮设计
- **主按钮**：
  - 尺寸：80px × 80px（圆形）
  - 颜色：橙色渐变
  - 圆角：24px
  
- **操作按钮**：
  - 尺寸：64px × 64px（方形）
  - 颜色：蓝色 #eff6ff
  - 圆角：16px

### 5.4 动画效果
- **浮动动画**：6秒循环，上下浮动
- **脉冲动画**：1.5秒循环
- **弹跳动画**：2秒循环
- **彩带动画**：2.5-3.5秒循环
- **过渡动画**：0.15秒 ease-out

## 6. 关卡系统

### 6.1 关卡结构
- **章节划分**：按章节组织关卡
- **关卡节点**：圆形按钮
- **星级评定**：1-3星

### 6.2 关卡进度

- **解锁状态**：根据进度解锁新关卡
- **特殊关卡**：带特殊标记
