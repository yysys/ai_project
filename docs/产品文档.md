# 猪了个猪 - 抖音小程序产品文档

## 1. 产品概述

### 1.1 产品简介
"猪了个猪"是一款轻松休闲的点击类抖音小程序，玩家通过点击屏幕上的猪猪，让它们沿着直线跑动，完成各种有趣的任务和挑战。游戏采用可爱的卡通画风，为玩家带来愉悦的游戏体验。

### 1.2 产品定位
- **目标用户**：18-35岁的休闲游戏玩家
- **游戏类型**：休闲益智类点击游戏
- **平台**：抖音小程序
- **核心玩法**：点击 + 直线跑动 + 策略思考

### 1.3 产品特色
- 可爱的卡通画风，吸引年轻用户
- 简单易上手，点击即可操作
- 丰富的关卡设计，保持游戏新鲜感
- 社交分享功能，增加传播性
- 轻松治愈的游戏体验

## 2. 游戏玩法

### 2.1 核心玩法
- 屏幕上有许多可爱的猪猪
- 玩家点击猪猪，猪猪会沿着直线跑动
- 猪猪跑动过程中会触发各种事件
- 玩家需要策略性地选择点击顺序和时机
- 完成关卡目标即可过关

### 2.2 游戏规则
1. **基本规则**
   - 点击猪猪，猪猪开始沿直线跑动
   - 猪猪跑动方向可以是上下左右四个方向
   - 猪猪跑动速度固定，不可改变
   - 猪猪跑动到边界或障碍物会停止

2. **胜利条件**
   - 所有猪猪都到达指定目标位置
   - 收集到指定数量的道具
   - 完成关卡设定的任务目标

3. **失败条件**
   - 猪猪跑动到危险区域
   - 超过时间限制
   - 猪猪数量不足完成任务

### 2.3 关卡设计
- **初级关卡**：猪猪数量少，路线简单
- **中级关卡**：猪猪数量增加，路线复杂
- **高级关卡**：猪猪数量多，路线复杂，有障碍物
- **特殊关卡**：限时模式、收集模式、挑战模式

### 2.4 猪猪类型
1. **普通猪猪**
   - 基础猪猪，点击即可跑动
   - 无特殊能力

2. **快速猪猪**
   - 跑动速度更快
   - 适合快速完成任务

3. **跳跃猪猪**
   - 可以跳过障碍物
   - 适合复杂地形

4. **炸弹猪猪**
   - 碰到障碍物会爆炸
   - 可以清除障碍

5. **磁铁猪猪**
   - 可以吸引附近的道具
   - 适合收集任务

## 5. 关卡设计

### 5.1 关卡类型
1. **普通关卡**
   - 基础关卡，引导玩家熟悉玩法
   - 猪猪数量：3-10只
   - 障碍物：无或少
   - 目标：让所有猪猪到达指定位置

2. **障碍关卡**
   - 引入障碍物，增加难度
   - 猪猪数量：5-15只
   - 障碍物：中等数量
   - 目标：避开障碍物到达目标

3. **收集关卡**
   - 收集道具类型的关卡
   - 猪猪数量：5-20只
   - 道具：星星、金币、宝石
   - 目标：收集指定数量的道具

4. **限时关卡**
   - 时间限制的关卡
   - 猪猪数量：10-30只
   - 时间：30秒-3分钟
   - 目标：在规定时间内完成任务

5. **挑战关卡**
   - 高难度关卡
   - 猪猪数量：20-50只
   - 障碍物：大量
   - 目标：完成复杂任务

### 5.2 关卡元素
1. **猪猪**
   - 普通猪猪
   - 快速猪猪
   - 跳跃猪猪
   - 炸弹猪猪
   - 磁铁猪猪

2. **障碍物**
   - 石头：阻挡猪猪
   - 水坑：猪猪需要跳跃
   - 陷阱：猪猪碰到会失败
   - 弹簧：改变猪猪方向

3. **道具**
   - 星星：收集得分
   - 金币：收集获得金币
   - 宝石：收集获得宝石
   - 钥匙：解锁特殊区域

4. **目标**
   - 传送门：猪猪到达传送门过关
   - 终点线：猪猪到达终点线过关
   - 收集箱：收集指定数量道具过关

### 5.3 关卡难度曲线
- **第1-10关**：新手引导，熟悉基本玩法
- **第11-30关**：引入障碍物，增加难度
- **第31-60关**：引入特殊猪猪，丰富玩法
- **第61-100关**：限时关卡，考验反应速度
- **第101关+**：挑战关卡，考验策略思考
